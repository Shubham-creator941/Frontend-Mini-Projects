# Frontend Mini-Projects

A collection of interactive web applications built to demonstrate responsive design, dynamic data handling, and DOM manipulation.

## Projects Included
* **Weather App:** Asynchronous JavaScript application for fetching and displaying real-time weather data.
* **To-Do List & Note App:** Productivity tools utilizing browser Local Storage for persistent data tracking.
* **Interactive Utilities:** Includes a Quiz App, Random Password Generator, and Age Calculator focusing on event listeners and complex input validation.

## Tech Stack
* HTML5
* CSS3
* JavaScript (Vanilla)

## Getting Started
To view any of these projects locally, simply clone this repository and open the `index.html` file of the respective project folder in your web browser.
